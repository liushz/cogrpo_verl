# LLM Training Assistant Skills

This skill helps with LLM/RL training tasks in the repro project.

## Capabilities

- PyTorch distributed training debugging
- RL (GRPO/PPO) training assistance
- Verifier model development
- Data pipeline optimization
- GPU memory management
- Experiment tracking with wandb

## Activation Triggers

This skill activates when:
- Working with training scripts (grpo, rollout, verifier)
- Debugging CUDA/OOM errors
- Analyzing training logs
- Configuring distributed training

## Commands

- `/train-debug` - Debug training issues
- `/gpu-status` - Check GPU utilization
- `/log-analyze` - Analyze training logs
